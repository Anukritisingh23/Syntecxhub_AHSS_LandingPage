# AHSS Landing Page

## Project Description
AHSS Landing Page is a UI/UX design project created in Figma for a hiking guide platform. 
The design focuses on clean layouts, typography hierarchy, reusable components, and a 
clear call-to-action to improve user engagement.

## Tools Used
- Figma
- Auto Layout
- Components
- Prototyping

## Key Features
- Clean and responsive landing page layout
- Structured sections: Hero, Features, CTA, Footer
- Reusable UI components
- CTA button prototyping

## Internship Program
This project was completed as part of the UI/UX Design Internship at Syntecxhub.
